Welcome to the Aircraft Models API

A simple REST API built with Express & Node.js that provides aircraft models, their IATA (International Air Transport Association) / ICAO (International Civil Aviation Organization) codes, manufacturer, engine type, and number of engines.

ðŸ“‹ Get All Models

Retrieve the full list of aircraft models.

GET /api/
Examples:
cURL: curl -X GET https://api-aircraft-models.vercel.app/api/
JavaScript (Fetch):
fetch('https://api-aircraft-models.vercel.app/api/')
.then(response => response.json())
.then(data => console.log(data));
Python:
import requests
response = requests.get('https://api-aircraft-models.vercel.app/api/')
print(response.json())
C#:
using System.Net.Http;
using System.Threading.Tasks;

HttpClient client = new HttpClient();
HttpResponseMessage response = await client.GetAsync("https://api-aircraft-models.vercel.app/api/");
string responseBody = await response.Content.ReadAsStringAsync();
Console.WriteLine(responseBody);
